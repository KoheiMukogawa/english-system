# 京大院試ドリルアプリ（app-kyoto）設計書

作成日: 2026-07-13
対象試験: 京都大学大学院（エネルギー科学系）入試の「英語」および「論述」
想定受験日: 2026年8月上旬（約3週間後）

## 1. 背景・目的

航空大学校の試験が終わり、次に京大院試（約3週間後）を控える。過去問6年分
（2019〜2025）が `kyoto-uni/` に投入され、Gemini による書き起こし
（`kyoto-uni/gemini-code-1783936002961.md`）も揃っている。

過去問分析から、この試験は一般入試英語ではなく、**エネルギー・環境・気候変動を
主題とする大学院入試**であることが判明した。英語と論述を半々で対策する方針。

目的は、過去問を反復演習できる**スマホ前提のドリルアプリ**を新設し、
和訳・和文英訳・補充・論述を効率的に回せるようにすること。

## 2. 出題傾向（分析結果）

### 英語（毎年ほぼ A・B・C の3題構成）
- **A**: 長文読解（硬め/古典寄り）— 下線部和訳・前置詞補充・整序英作文・語義選択・内容説明
- **B**: 長文読解（科学論文・レポート系）— 語彙補充・下線部和訳・内容説明（字数指定）
- **C**: 和文英訳（エネルギー政策文書の全文英訳）— **毎年必出**

設問タイプの配点重心（高→低）:
1. 下線部和訳（英→和）… 最頻・最重要
2. 和文英訳（和→英）… 毎年必出
3. 前置詞補充／語彙補充（選択肢群から）… 頻出
4. 整序英作文 / 内容説明（字数指定）… 中頻度
5. 語義選択（4択）／数字・論理 … 少数

テーマ・頻出語彙: renewable / carbon neutral / greenhouse gas / decoupling /
feed-in tariff / power mix / emissions / anthropogenic / circular economy /
biomass など。2024-25では確率パズル（モンティ・ホール）等、論理系にも拡大。

### 論述（別科目・計算＋日本語小論文）
CO₂濃度計算・エネルギー自給率・炭素循環などの**計算問題**と、200〜800字の
**日本語小論文**（キーワード指定あり）。テーマはプラスチック汚染・原子力・
リチウム・カーボンニュートラル・林業・SDGs 等。

## 3. スコープ（意思決定の記録）

- 対象科目: **英語と論述を半々**
- アプリ: **京大用の新アプリを別途新設**（`app/` を複製、`app-met/` の前例に倣う）
- 論述の扱い: **AI採点の小論文モードを新設**（計算問題は模範解法の閲覧のみ）
- 英語 v1 のモード: **核（和訳＋和文英訳＋補充四択）に絞る**
  （整序英作文・内容説明は v2 に後回し）

## 4. アーキテクチャ

`app/index.html`（1068行）の既存エンジンを土台に複製・拡張する。既存エンジンは
以下を既に備えており、そのまま流用する:

- SRS（間違い箱 `box` / 復習期日 `due`、STATS 単位で管理）
- 和訳モード `type:"tr"`（5段階自己採点 + 回答履歴 TRLOG 直近5回）
- Claude採点（Anthropic API をブラウザ直叩き、`claude-opus-4-8`、端末ローカルにキー保存）
- PWA（manifest）/ Gist 同期 / IndexedDB + localStorage 永続化

### ファイル構成
```
app-kyoto/
  index.html          # app/index.html を複製し、モードを追加
  questions.json      # 京大用データ（英語＋論述）
  manifest.webmanifest
  icons/
```

### モード一覧
| モード | 由来 | 実装 | 採点 |
|---|---|---|---|
| 和訳（英→和） | 既存 `type:"tr"` | そのまま | 5段階自己採点＋Claude採点 |
| 補充四択（前置詞・語彙・語義） | 既存 4択 | そのまま | 自動正誤 |
| 和文英訳（和→英） | 新 `type:"en"` | tr画面を反転（問=日本語／解答=英語） | 5段階自己採点＋Claude採点（プロンプト差替） |
| 論述（小論文） | 新 `type:"essay"` | 長文入力＋キーワード/字数表示 | Claude採点（ルーブリック）＋模範解答閲覧 |

計算問題は模範解法の閲覧のみ（自動採点なし）。

## 5. データ構造（questions.json）

既存フィールド（`id, part, cat, q, a, d, e, type, ety`）を踏襲し、京大向けに
意味を再割り当てし、2種のフィールドを追加する。

**共通**
- `id`: 通し番号
- `part`: **出典（年度・大問）**。例 `"2023-A"`, `"2021-C"`。年度／大問での絞り込み演習に使う
- `cat`: **出題タイプ or テーマ**。例 `"和訳・実戦"`, `"和文英訳"`, `"前置詞補充"`, `"論述・エネルギー"`
- `q`: 問題文、`a`: 模範解答、`e`: 解説
- `type`: `"tr"`(和訳) / `"en"`(和文英訳) / `"essay"`(論述) / 無指定(四択)

**追加フィールド**
- `hint`: 語注（和文英訳・読解の注記語）。任意
- `kw`: 論述のキーワード配列。任意
- `limit`: 論述の字数目安（整数）。任意

**四択**: 既存どおり `d`（誤答選択肢の配列）を使用。

### 例
```jsonc
// 和訳（英→和）: 既存 tr そのまま
{ "id":301, "part":"2020-A", "cat":"和訳・実戦", "type":"tr",
  "q":"If we have force in any one of its forms ...(下線部ア)",
  "a":"もし我々が力を…（訳例）", "e":"turn A into B＝… / 構文解説" }

// 和文英訳（和→英）: 新 type "en"
{ "id":401, "part":"2024-C", "cat":"和文英訳", "type":"en",
  "q":"大気汚染は現在、深刻な死亡リスク要因となっており、2021年には世界で810万人が死亡している。",
  "a":"Air pollution is now a serious mortality risk factor, ...（英訳例）",
  "e":"『死亡リスク要因』=mortality risk factor / 時制の選択", "hint":"malnutrition＝栄養失調" }

// 論述（小論文）: 新 type "essay"
{ "id":501, "part":"2022-C", "cat":"論述・エネルギー", "type":"essay",
  "q":"2050年カーボンニュートラル実現に向け電力部門に求められる取組を、2030年まで/以降に分けて600字程度で論じなさい。",
  "kw":["再生可能エネルギー","原子力再稼働","系統","調整力"],
  "limit":600,
  "a":"【模範骨子】…（論点）", "e":"評価観点：①…②…③…" }
```

## 6. AI採点フロー（type別分岐）

既存 `aiGrade()`（`app/index.html:324` 付近）は和訳固定プロンプト。これを `type`
で分岐させ、既存の呼び出し基盤（API直叩き・点数正規表現抽出→自己採点/SRS反映・
読めなければ手動採点フォールバック）はそのまま流用する。

| type | 採点プロンプト（要旨） | 出力 |
|---|---|---|
| `tr` | 既存のまま「英文和訳を添削【英文】【訳例】【学生の訳】」 | 点数＋添削 |
| `en` | 「和文英訳を添削。文法・語法・自然さ・専門語の妥当性【和文】【英訳例】【学生の英訳】」 | 点数＋添削＋better表現 |
| `essay` | 「小論文をルーブリック採点。観点①設問要求充足②論理構成③根拠/数値④キーワード網羅⑤字数【設問】【キーワード】【字数】【模範骨子】【学生の答案】」 | 観点別点＋総評＋改善案 |

- `en`・`essay` の入力欄は既存 `trIn` を流用。essay は複数行入力＋字数カウンタを追加。
- API キーは端末ローカル保存（既存挙動）。オフライン時は自己採点のみで成立。

## 7. コンテンツ計画

書き起こし済み6年分から `questions.json` 化。模範解答・解説は Claude が作成する。

### 初期投入量（v1 目安）
| モード | 出典 | 概算件数 |
|---|---|---|
| 和訳(tr) | 各読解の下線部（Jevons ア/ウ、デカップリング ウ、UNEP エ/オ、Covid ア、人工物量 イ、モンティ ア/イ/ウ 等） | 10〜12 |
| 和文英訳(en) | 大問C 4本（エネルギー基本計画・エネルギー未来・FIT・大気汚染）を文単位に分割 | 15〜18 |
| 補充四択 | 前置詞補充(Covid ①〜⑥)・語彙選択(あ〜お群)・語義選択(contingent) | 20〜25 |
| 論述(essay) | プラ汚染・自給率・原子力・リチウム・カーボンニュートラル・林業・SDGs・石油需要 等 | 12〜15 |
| 計算（閲覧のみ） | CO₂濃度・マイクロプラ個数・自給率 | 3 |

### 段取り（3週間に同期）
- Phase 1: 和訳＋和文英訳（最優先・AI採点即活用）→ Week 1 開始で回せる
- Phase 2: 補充四択 → Week 1 後半
- Phase 3: 論述 → Week 2
- Phase 4（任意/v2）: 整序・内容説明モード → Week 3 に余力があれば

### 品質の注意
論述の「模範骨子」は Claude 作成のため、論点の網羅・方向性の担保に用途を留める。
模範解答の丸暗記は避け、採点も骨子一致ではなく観点ルーブリックで評価する。

## 8. スコープ外（YAGNI）

- 整序英作文モード・内容説明（字数）モード … v2（Phase 4）に後回し
- 計算問題の自動採点 … 模範解法の閲覧のみ
- 航大アプリ（`app/`）との統合 … 別アプリとして独立させる

## 9. 学習プラン（本番 ≈ 8/3 想定）

- Week 1: 和訳の型づくり＋頻出語彙。過去問 A・B の下線部和訳を tr でスマホ演習化。
- Week 2: 和文英訳（白書の定型表現）を反復。補充四択を回す。論述の型づくり。
- Week 3: 年度単位の通し演習（時間計測）＋弱点補強。論述の観点別仕上げ。
