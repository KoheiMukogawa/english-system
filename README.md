# 航大英語 大問I ドリル

航空大学校 一次試験・英語「大問I」（短文空所補充・4択）対策の学習アプリ。
全171問を間隔反復（Leitner方式）で効率よく復習できます。

## 構成

```
english-system/
├── app/                      # アプリ本体（このフォルダを丸ごとデプロイする）
│   ├── index.html            # アプリ（単一ファイル・ビルド不要）
│   ├── questions.json        # 問題データ171問（正データはルートのものと同一）
│   ├── manifest.webmanifest  # ホーム画面追加用
│   └── icons/                # アプリアイコン
├── questions.json            # ★唯一の正データ源（改変禁止）
├── 航大英語_大問I_ドリルアプリ.html   # 旧プロトタイプ（参考用）
└── CLAUDE_CODE_引き継ぎプロンプト.md  # 要件定義
```

## ローカルで起動する

```bash
cd app
python3 -m http.server 8000
# → ブラウザで http://localhost:8000 を開く
```

`index.html` を直接ダブルクリックすると `questions.json` の読み込みに失敗するため、
必ず上記のようにローカルサーバー経由で開いてください。

## デプロイ（GitHub Pages）

1. GitHubにリポジトリを作成し、このフォルダを push
2. リポジトリの Settings → Pages → Source を「Deploy from a branch」にし、
   ブランチ `main`・フォルダ `/app` は選べないため、`app/` の中身を `docs/` にコピーするか
   `main` ルート直下に置く（もしくは Actions でデプロイ）
3. 発行された URL を iPhone の Safari で開き、共有メニュー →「ホーム画面に追加」

Netlify / Vercel の場合は `app/` を公開ディレクトリに指定するだけです。

## 主な機能

- **スマート復習** … 復習期限が来た問題 → 未学習 → その他 の優先順で20問出題
- **間隔反復（Leitner）** … 正解するたび出題間隔が 1日 → 3日 → 7日 → 14日 と延長。
  間違えると間隔リセットで「復習待ち」に戻る。box3以上でマスター扱い
- **間違えた問題だけ** … 一度誤答してまだマスターしていない問題のみ出題
- **パート別練習 / 全問ランダム25問**
- **統計** … 復習待ち・未学習・マスター・累計正答率・連続学習日数（streak）
- **進捗の永続化** … IndexedDB + localStorage の二重保存。
  旧プロトタイプ（localStorage `kdai_stats_v1`）の進捗は初回起動時に自動移行

## 進捗のエクスポート / インポート（機種変更・バックアップ）

1. メニュー最下部の **「データ管理（バックアップ・引き継ぎ）」** を開く
2. **エクスポート**: 「ファイルをダウンロード」でJSONファイル保存、
   または「コードをコピー」でクリップボードへ
3. **インポート**: 新しい端末で同じ画面を開き、ファイルを選択するかコードを貼り付けて取り込む
   （現在の進捗は上書きされるので注意）

## データについて

- `questions.json`（ルート）が唯一の正データ源。**内容は改変しないこと**
- 問題を修正する場合はルートの `questions.json` を直し、`app/questions.json` にコピーする
- スキーマ: `{id, part, cat, q, a, d[3], e}`（`q`の`( )`が空所、`a`が正解、`d`が誤答3つ、`e`が解説）
